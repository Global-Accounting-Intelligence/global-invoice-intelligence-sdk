# Global Invoice Intelligence SDK

Public SDK, schemas, examples, and connector starters for Global Invoice Intelligence.

## What This Repository Contains

- Public JSON schemas.
- OpenAPI specification.
- Python SDK.
- JavaScript SDK.
- ERPNext connector starter.
- Odoo connector starter.
- Generic REST connector starter.
- Fake sample invoices and match results.

## What This Repository Does Not Contain

- Private matching engine.
- Trust engine.
- Learning engine.
- Material scoring implementation.
- Real customer invoices.
- Secrets.
- Internal deployment information.

## Quick Start

1. Read `docs/GETTING_STARTED.md`.
2. Review `openapi/openapi.yaml`.
3. Validate example invoice against `schemas/invoice.schema.json`.
4. Use Python or JavaScript SDK to submit invoice data.
