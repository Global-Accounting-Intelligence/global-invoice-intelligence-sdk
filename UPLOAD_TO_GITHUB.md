# Upload Instructions

## Repository

`global-invoice-intelligence-sdk`

## Recommended Method

1. Extract this ZIP file.
2. Open the extracted folder.
3. Copy all contents into the matching GitHub repository local folder.
4. Run:

```bash
git add .
git commit -m "Initial project structure and documentation"
git push
```

## Safety

Before pushing, verify:

- No `.env` file.
- No real customer invoices.
- No API keys.
- No passwords.
- No private customer data.
