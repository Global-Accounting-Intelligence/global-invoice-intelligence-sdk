# Country Packs

Country Packs define country-specific invoice, tax, XML, QR, and validation rules.

Do not promote any auto-generated country pack to production without governance review.

Source trust ranking:

1. Official tax authority sources.
2. Official e-invoice schemas.
3. Official government APIs.
4. Recognized standards such as UBL/Peppol.
5. Certified implementation guides.
6. Expert-reviewed input.
7. Customer-submitted examples.
8. Repeated multi-tenant patterns.
9. AI-generated interpretation.
